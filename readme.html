Project name: Coockook Set up: 1. Download Wordpress 2. Set up enviroment (WAMP, LAMP, DOCKER) 3. Create projects domain 4. Create database in PHPMyADMIN with name "wordpress" 5. Inport database file from folder db/wordpress.sql Activate Theme Coockbook: Log in Go to the appearance-> themes -> activate cookcbook theme Details of theme Coockbook: On first page user (if he is login) he can add Main page widget text to give short intro of the company, also he can add images widget to display team members on about us page, there is also sidebar widget to be added for search field on single post page/ User can add blogs and recipies that will be displayed on Blog and Recipies page, latest will be on front page.
